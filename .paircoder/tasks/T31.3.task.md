# T31.3 — Live Code Terminal

The core viewer addition. Shows part function calls synchronized with replay playback.

## Implementation
Create viewer/js/code-terminal.js:
- initCodeTerminal(replay): create DOM panel in the right side
- updateCodeTerminal(carData, tick, replay): render current tick's calls
- Color-coded: green (ok), yellow (clamped), red (glitch/error)
- Show: part_name() → output [status]
- Scrolling log that follows current tick
- Click a car in timing tower to switch which car's code is shown

## Layout
Add right panel to dashboard.html for the code terminal.
Wire into main.js render loop.

## AC
- Code terminal shows part calls for selected car
- Colors indicate status
- Synchronized with replay playback (play/pause/scrub)
- Handles missing call_logs gracefully (shows "No call data")
