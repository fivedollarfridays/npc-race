# T47.1 — Tiered Car Loading
Classify 19 rivals into tiers. load_all_cars accepts tier filter.

## Tiers
- Rookies (4): Tortoise, RustBucket, PaperWeight, SteamRoller
- Midfield (6): IronSide, CrossWind, FoxFire, DriftKing, Gambler, Chameleon
- Frontrunners (5): GlassCanon, Berserker, NightFury, Vortex, Phantom
- Veterans (4): BrickHouse, Silky, GooseLoose, SlipStream

## API
- `load_tier_cars(tier: str) -> list[dict]` — "rookie", "midfield", "front", "full"
- "full" returns all 19
