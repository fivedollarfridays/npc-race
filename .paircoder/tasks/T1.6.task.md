# Objective

Update play.py CLI to support named track selection and track listing.

# Implementation Plan

1. Add `--track NAME` argument to argparse
2. Add `--list-tracks` flag that prints available tracks and exits
3. When --track specified, pass track_name to run_race
4. When --track "random", pick a random track
5. When --track specified, ignore --seed
6. Update help text
7. Write tests

# Acceptance Criteria

- [ ] `python play.py --track monza` runs race on Monza
- [ ] `python play.py --track random` picks random track
- [ ] `python play.py --list-tracks` prints all 20 tracks with name, country, character
- [ ] `python play.py --seed 42` still works (procedural, no track_name)
- [ ] --track ignores --seed when both specified
- [ ] Invalid --track name gives clear error message
- [ ] Tests cover all flag combinations
