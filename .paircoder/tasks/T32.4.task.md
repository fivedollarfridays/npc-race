# T32.4 — Submit Command

Add `npcrace submit` CLI command that validates and prepares results for submission.

## Implementation
- New CLI subcommand: `npcrace submit results.json`
- Validates: file exists, JSON parses, integrity hash valid
- Prints: summary of results (position, time, league, parts)
- For now: local validation only (no server upload)
- Prints: "Results verified. Ready for leaderboard submission."
- Future: will POST to a leaderboard API (Sprint 33)

## Why
The submit command is the player-facing interface for the product loop.
Even without a server, it validates their results file is well-formed
and untampered. Sprint 33 adds the actual upload target.

## AC
- `npcrace submit results.json` validates and prints summary
- Tampered results rejected: "Integrity check failed"
- Missing file: clear error message
- Valid results: "Verified. Position: P1, Time: 405.23s, League: F3"
