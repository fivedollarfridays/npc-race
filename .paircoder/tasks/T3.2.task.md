# Objective

Create engine/tire_model.py with three tire compounds featuring non-linear wear curves and cliff effect.

# Implementation Plan

1. Define COMPOUNDS dict: soft (base_grip 1.15, wear 0.00040, cliff 0.75), medium (1.00, 0.00020, 0.80), hard (0.88, 0.00012, 0.85)
2. compute_wear(current_wear, compound, throttle, curvature) — wear increases faster with higher throttle and curvature
3. compute_grip_multiplier(wear, compound) — non-linear: base_grip * max(0.3, 1 - (wear/cliff)^exponent). Sharp drop after cliff.
4. is_past_cliff(wear, compound) — boolean check
5. get_compound(name) — return compound config, default medium

# Acceptance Criteria

- [ ] SOFT gives highest grip fresh, degrades fastest
- [ ] HARD gives lowest grip fresh, lasts longest
- [ ] Grip drops >30% when wear exceeds cliff threshold
- [ ] All compounds return grip >= 0.3 minimum
- [ ] compute_wear increases monotonically
- [ ] Module < 200 lines, < 15 functions
