# T8.1 — Replay Enrichment for Dashboard

## Objective

Add missing fields to replay frames that the pit wall dashboard needs. Currently frames have speed/tire/fuel/drs/dirty_air but are missing gap_behind, last_lap_time, best_lap, sector times, tire age, pit count, and dirty air factor as a number.

## Fields to Add to Per-Frame Data

In `engine/replay.py`, add to each frame dict:

```python
"gap_behind_s": round(state.get("_gap_behind_s", 0.0), 3),
"last_lap_time": state.get("_last_lap_time"),       # float seconds or None
"best_lap_s": state.get("_best_lap_s"),              # float seconds or None
"tire_age_laps": state.get("tire_age_laps", 0),     # int
"pit_stops": state.get("pit_state", {}).get("pit_stops", 0),  # int
"dirty_air_factor": round(state.get("_dirty_air_grip", 1.0), 3),
```

## Fields to Compute in simulation.py

In `_step_car()`, after timing update, store on car state:
```python
state["_gap_behind_s"] = strat_state["gap_behind_s"]
state["_last_lap_time"] = (self.timings[state["name"]].lap_times[-1]
                           if self.timings[state["name"]].lap_times else None)
state["_best_lap_s"] = self.timings[state["name"]].best_lap
```

## Sector Time Events

When a sector completes (timing result says sector_completed=True), store the sector time:
```python
if t_result.get("sector_completed"):
    state["_last_sector_time"] = t_result["sector_time"]
    state["_last_sector_idx"] = t_result.get("current_sector", 0) - 1
```

Add to frame:
```python
"last_sector_time": state.get("_last_sector_time"),   # float or None (only on sector completion tick)
"last_sector_idx": state.get("_last_sector_idx"),      # 0, 1, or 2
```

Clear after recording: `state["_last_sector_time"] = None` (so it only appears on the completion tick).

## Tests

Add to `tests/test_timing_enrichment.py`:
