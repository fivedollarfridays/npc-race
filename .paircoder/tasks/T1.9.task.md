# Objective

Run all 5 seed cars on all 20 tracks. Verify no single car dominates all track types. Tune physics constants if needed. Create a balance test script.

# Implementation Plan

1. Create `tests/test_balance.py` -- runs all cars on representative tracks
2. Create `scripts/balance_report.py` -- generates win matrix (car x track)
3. Run matrix and analyze:
   - Power tracks should favor GlassCanon/BrickHouse
   - Technical tracks should favor Silky/GooseLoose
   - Balanced tracks should be competitive across cars
   - Long races (5+ laps) should favor tire conservers
4. If one car dominates, adjust physics constants
5. Document any physics changes

# Acceptance Criteria

- [ ] Balance test script runs all 5 cars on 5+ representative tracks
- [ ] No single car wins more than 60% of tracks
- [ ] Power tracks have different winners than technical tracks
- [ ] Win distribution documented in test output
- [ ] Any physics adjustments documented with before/after results
- [ ] Tests are repeatable (deterministic with seed)
