# T17.1 — WebSocket Server

## Objective

Create `engine/live_server.py` — an asyncio WebSocket server that runs a race simulation and streams frames to connected viewers in real-time.

## Design

**Protocol (JSON messages):**

Server → Client:
```json
{"type": "init", "track": [...], "track_width": 50, "track_name": "monza",
 "laps": 3, "car_count": 5, "ticks_per_sec": 30,
 "track_curvatures": [...], "track_headings": [...]}
```
```json
{"type": "frame", "cars": [{...}, {...}], "tick": 42}
```
```json
{"type": "results", "results": [...], "events": [...],
 "commentary": [...], "race_report": "..."}
```

Client → Server:
```json
{"type": "pause"}
{"type": "resume"}
{"type": "speed", "value": 2}
```

**Modes:**
- **Live mode**: run simulation, stream each frame after computing it
- **Replay mode**: load a saved replay JSON, stream frames at playback speed
