# T7.2 — Speed-Dependent Downforce/Grip

## Objective

In real F1, aerodynamic downforce is proportional to speed². This means fast corners (200+ km/h) have much more grip than slow corners (60 km/h). Currently NPC Race's grip is speed-independent — a car has the same cornering ability at 60 km/h as 250 km/h, which is fundamentally wrong.

This also makes the wing_angle setup slider more meaningful: high wing = more downforce = faster in high-speed corners but slower on straights.

## Research Basis

- Real F1: downforce = 0.5 * rho * Cl * A * v² → ~3.5x car weight at 300 km/h
- Speed Dreams / TORCS: speed-dependent grip coefficient
- Current NPC Race: grip is purely from GRIP stat + tire condition, no speed factor

## Implementation Plan

### Add to `engine/physics.py` (~15 LOC addition)

```python
DOWNFORCE_GRIP_FACTOR = 0.25   # Max aero grip bonus at reference speed
DOWNFORCE_REF_SPEED = 300.0    # km/h — speed at which aero grip is fully developed

def compute_aero_grip(speed: float, aero: float, wing_angle: float = 0.0) -> float:
    """
    Speed-dependent grip bonus from downforce.

    Returns additional grip multiplier (0.0-0.3 range).
    At 300 km/h with aero=1.0 and wing=+1.0: bonus ≈ 0.30
    At 60 km/h: bonus ≈ 0.012 (negligible)

    wing_angle from setup increases/decreases the effect:
    - wing_angle = +1.0: 1.15x downforce (more grip, more drag)
    - wing_angle = -1.0: 0.85x downforce (less grip, less drag)
    """
    speed_factor = (speed / DOWNFORCE_REF_SPEED) ** 2
    speed_factor = min(1.2, speed_factor)  # slight cap above ref speed
    wing_mult = 1.0 + wing_angle * 0.15  # from setup_model
    return DOWNFORCE_GRIP_FACTOR * aero * speed_factor * wing_mult
```

### Impact on physics

In `compute_target_speed()`, the grip_speed for corners becomes:
```python
effective_grip = grip * tire_grip_mult * temp_grip_mult + compute_aero_grip(speed, aero, wing_angle)
```

This means:
- **Slow hairpin (60 km/h)**: aero_grip ≈ 0.01 → mostly mechanical grip
- **Fast sweeper (200 km/h)**: aero_grip ≈ 0.11 → significant aero contribution
- **Full-speed curve (300 km/h)**: aero_grip ≈ 0.25 → aero-dominant grip
