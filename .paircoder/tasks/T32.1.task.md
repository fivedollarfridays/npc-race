# T32.1 — Results Summary Format

Define and implement a lightweight results format extracted from the full replay.

## Format: results_summary.json
```json
{
  "version": "1.0",
  "track": "monza",
  "laps": 5,
  "league": "F3",
  "timestamp": "2026-03-23T12:00:00Z",
  "cars": [
    {
      "name": "MyCarProject",
      "position": 1,
      "total_time_s": 405.23,
      "best_lap_s": 80.12,
      "lap_times": [81.5, 80.8, 80.3, 80.12, 82.5],
      "reliability_score": 0.94,
      "league": "F3",
      "loaded_parts": ["gearbox", "cooling", "strategy"]
    }
  ],
  "integrity": "sha256:abc123..."
}
```

## Implementation
- engine/results.py: `generate_results_summary(replay, cars, league)` -> dict
- Extract essential fields from replay (not frames, not call_logs)
- Include car metadata: reliability_score, loaded_parts, league
- Add timestamp
- File is small (<10KB vs replay at 10MB+)

## AC
- Results summary generated from a race replay
- Contains all fields listed above
- File size under 50KB for a 5-lap 6-car race
