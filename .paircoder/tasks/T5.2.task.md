# T5.2 ‚Äî DRS System

## Objective

Model the Drag Reduction System used in F1: a movable rear wing element that reduces drag on designated straights when a car is within 1 second of the car ahead. Create `engine/drs_system.py`, add DRS zone data to named tracks, and expose activation state to strategies.

## Implementation Plan

### New file: `engine/drs_system.py` (~80 LOC)

```python
# DRS zone: (start_pct, end_pct) of lap distance ‚Äî fraction 0.0‚Äì1.0
DRS_GAP_THRESHOLD = 1.0  # seconds ‚Äî activation eligibility

def get_drs_zones(track_dict: dict) -> list[tuple[float, float]]:
    """Return DRS zones from track data, or [] if none defined."""

def is_in_drs_zone(distance_pct: float, zones: list[tuple[float, float]]) -> bool:
    """True if current lap position is inside any DRS zone."""

def drs_speed_multiplier(in_zone: bool, drs_active: bool) -> float:
    """Return speed multiplier: 1.05 when DRS open, 1.0 otherwise."""

def update_drs_state(car_state: dict, drs_requested: bool,
                     in_zone: bool, gap_ahead_s: float,
                     lap_changed: bool) -> dict:
    """
    Compute next DRS state:
    - Reset availability on lap change
    - Activate if: requested + in_zone + gap_ahead_s <= GAP_THRESHOLD + available
    - Deactivate when leaving zone
    Returns updated car_state subset: {drs_available, drs_active}
    """
```

### Track data: add `drs_zones` to named tracks

Update these tracks in `tracks/` (5‚Äì6 tracks with real DRS zones):

- `bahrain.py` ‚Äî zones at ~0.08‚Äì0.20 and ~0.60‚Äì0.72
- `silverstone.py` ‚Äî zone at ~0.62‚Äì0.78
