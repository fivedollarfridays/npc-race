**Silky** — brake temp aware:
- If `brake_temp > 650`: reduce throttle to 0.8 (cool brakes before next corner)
- If `brake_temp > 750`: switch to "conserve" engine mode

**BrickHouse** — heavy braker, ERS harvester:
- Default to "harvest" mode (heavy car generates more braking energy)
- Switch to "attack" in final 2 laps
