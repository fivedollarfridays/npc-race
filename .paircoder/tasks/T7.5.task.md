### Wire aero grip into `compute_target_speed()`

In engine/physics.py `compute_target_speed()`:
```python
from .physics import compute_aero_grip
aero_grip = compute_aero_grip(current_speed, aero, wing_angle)
effective_grip = grip * tire_grip_mult * temp_grip_mult + aero_grip
```

This requires knowing current speed when computing target speed. May need to use previous tick's speed as the input (standard approach — speed from last tick determines this tick's cornering grip).
