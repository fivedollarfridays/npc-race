# T14.2 — Turbo Whistle Synthesis

## Objective

Add turbo whistle effect to `sound-engine.js`. When a car decelerates (speed drops), play a descending whistle sound. More dramatic on harder braking.

## Design

**Turbo whistle:** High-pitched sine wave (2-4 kHz) that descends as the car decelerates.
- Trigger: speed drop > 30 km/h between frames
- Pitch: starts at 4 kHz, descends to 2 kHz over 0.3s
- Volume: proportional to speed drop magnitude
- One global turbo oscillator, triggered for nearest car

**Implementation:**
- Add `_turboOsc` sine oscillator + gain node
- `triggerTurboWhistle(speedDrop)` — ramp pitch down, fade volume
- Call from `updateSound` when selected car decelerates

## Changes to `viewer/js/sound-engine.js`

1. Add `_turboOsc`, `_turboGain` nodes in `initAudio()`
2. Add `triggerTurboWhistle(speedDrop)` function
3. Wire into `updateSound` — detect speed drops, trigger whistle

## Tests

- `test_turbo_whistle_function` — JS has `triggerTurboWhistle`
- `test_turbo_oscillator_created` — JS creates turbo oscillator node
