# T34.2 — Fast Mode in RaceSim

Add `fast_mode=False` to `RaceSim.__init__`. In fast mode: record frames
at 1Hz (every 30 ticks), feed LapAccumulator every tick.

## Key constraint: Physics IDENTICAL in both modes. Only storage changes.
