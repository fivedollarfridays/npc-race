# T47.3 — Progression System
Track player progress: ghost levels beaten, tiers unlocked.

## State file: ~/.npcrace/progress.json
```json
{
  "ghost_completed": {"monza": 5, "monaco": 3},
  "tier": "midfield",
  "races_won": 2,
  "tracks_completed": ["monza"]
}
```

## Unlock rules
- Beat Ghost Level 5 on any track → unlock "midfield" tier
- Win a midfield race → unlock "front" tier
- Win a frontrunner race → unlock "full" grid
- Ghost optional after first track completion
