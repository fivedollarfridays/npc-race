# Objective

Create engine/pit_lane.py with pit stop state machine: entry, stationary, exit, compound swap.

# Implementation Plan

1. PIT_STATES: racing, pit_entry, pit_stationary, pit_exit
2. PIT_SPEED_LIMIT = 80 km/h, PIT_STOP_DURATION_TICKS = 660 (~22s at 30 tps)
3. create_pit_state() — initial state dict
4. request_pit_stop(pit_state, compound) — queue pit stop
5. update_pit_state(pit_state, car_distance, track_length, dt, tps) — state machine tick
6. is_in_pit(pit_state) — boolean
7. get_speed_limit(pit_state) — returns limit or None
8. complete_pit_stop(pit_state) — returns (updated_state, new_compound), resets wear

# Acceptance Criteria

- [ ] State transitions: racing → pit_entry → pit_stationary → pit_exit → racing
- [ ] Speed limited during pit_entry and pit_exit
- [ ] Stationary phase lasts ~22 seconds
- [ ] Tire compound swapped on completion
- [ ] pit_stops counter increments
- [ ] Cannot pit while already in pit
- [ ] Module < 200 lines, < 15 functions
