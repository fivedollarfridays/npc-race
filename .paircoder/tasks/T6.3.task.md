# T6.3 — Tire and Fuel Recalibration

## Objective

After T6.2 fixes speed scale, tire wear/temperature and fuel consumption need retuning. Currently: tires wear out in ~4 laps (should be 15-40), tire temps max at 65°C (should be 80-110°C), fuel shows 0 consumed. Retune to F1-realistic values.

## Implementation Plan

### Tire wear rates (engine/tire_model.py)

Current rates were tuned for 7-second laps. With 80-second laps (~12× longer), tires would disintegrate in half a lap. Scale down by ~10×:

```python
# Current → New
"soft":   {"wear_rate": 0.00040 → 0.000040}   # ~25 laps to 1.0 wear
"medium": {"wear_rate": 0.00020 → 0.000018}   # ~35 laps to 1.0 wear
"hard":   {"wear_rate": 0.00015 → 0.000012}   # ~50 laps to 1.0 wear
```

Verify: at 30 ticks/sec × 80s/lap × 0.000040 × (0.5 + 0.5*0.8) × (1 + 0.1*5) = wear per lap.
Target: soft lasts 15-25 laps, medium 25-35, hard 35-50.

### Tire temperature (engine/tire_temperature.py)

Current heat generation scaled for insane speeds. With realistic 250 km/h average:
- Heat gen needs to push tires to 80-110°C operating range within 2-3 laps
- Dissipation should stabilize at ~90°C under typical load

Adjust constants:
```python
# heat_generation: (throttle * THROTTLE_HEAT + curvature * CORNER_HEAT + abs(lateral) * LATERAL_HEAT) * dt
# Target: reach 80°C from 20°C in ~2 laps = ~160s = ~4800 ticks
# Need: 60°C / 4800 ticks = 0.0125°C/tick average net heat
# At equilibrium (90°C), dissipation = generation

THROTTLE_HEAT = 3.0        # was 2.0 — more heat at full throttle
CORNER_HEAT = 12.0         # was 8.0 — cornering is primary heat source
LATERAL_HEAT = 2.5         # was 1.5

# Dissipation: tuned so equilibrium at ~90°C under typical conditions
PASSIVE_RATE = 0.025       # was 0.036 — less aggressive cooling
SPEED_COOL_FACTOR = 0.0001 # was 0.00002 — slightly more airflow cooling at lower speeds
```
