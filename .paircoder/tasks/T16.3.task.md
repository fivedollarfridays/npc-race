# T16.3 — Car Development

## Objective

Create `engine/car_development.py` — cars earn development points from race finishes and spend them to upgrade stats between races. Budget cap prevents maxing everything.

## Design

**Development points earned per race:**
- P1: 10 dev points, P2: 7, P3: 5, P4: 3, P5: 2, P6-P10: 1, DNF: 0

**Stat upgrades:**
- Each stat (POWER/GRIP/WEIGHT/AERO/BRAKES) can be upgraded by spending dev points
- Cost: 5 dev points per +1 stat point
- Budget cap still applies: total stats ≤ 100
- Maximum +10 upgrade per stat per season

**Development state:** persists across races in a season, resets between seasons.
