# T5.4 — Simulation Integration

## Objective

Wire all three Tier 2 modules (tire temperature, DRS, car setup) into `engine/simulation.py`. Expose new state fields to strategies; handle new return fields from strategies; apply physics modifiers each tick.

## Scope Warning

`simulation.py` is currently ~353 lines / 14 functions (limits: 400 lines error, 15 functions error). Adding three systems risks hitting both limits. Must use extraction discipline: delegate all per-system logic to the new modules; simulation.py only calls them.

## Implementation Plan

### Car state initialization (`RaceSim.__init__`, car state dict ~line 60–73)

Add:
```python
"tire_temp":      20.0,   # °C — cold at race start
"drs_available":  True,   # reset each lap
"drs_active":     False,
"setup":          apply_setup(cs["stats"], cs["setup"]),  # precomputed effective stats
```

### `_apply_physics()` changes

1. **Tire temperature** (after tire wear update):
   ```python
   heat = heat_generation(throttle, curvature, lateral, dt)
   heat *= cs["setup"]["temp_rate_mult"]          # suspension effect
   cool = heat_dissipation(cs["tire_temp"], speed, dt)
   cs["tire_temp"] = update_tire_temp(cs["tire_temp"], heat, cool)
   temp_grip = tire_temp_grip_factor(cs["tire_temp"], cs["tire_compound"])
   # Multiply into existing grip: effective_grip *= temp_grip
   ```

2. **DRS** (before target speed calculation):
   ```python
   distance_pct = cs["distance"] % self.track_length / self.track_length
   in_zone = is_in_drs_zone(distance_pct, self.drs_zones)
   cs = update_drs_state(cs, decision.get("drs_request", False),
                          in_zone, state["gap_ahead_s"], lap_changed)
   speed_mult *= drs_speed_multiplier(in_zone, cs["drs_active"])
   ```

3. **Setup** (effective stats lookup):
   - Replace direct `cs["power"]` / `cs["aero"]` references with `cs["setup"]["effective_power"]` etc.
   - Computed once at init, not per-tick (setup is pre-race, not dynamic)
