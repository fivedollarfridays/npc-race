# T23.1 — Lock Sensitivity Test Script

Finalize `scripts/sensitivity_test.py` so it:
1. Runs a 1-lap Monza race with all default parts (baseline)
2. For each of 9 codeable parts, runs with ONE optimized variant + rest default
3. Runs with ALL optimized variants together
4. Prints a clear table: part name, default time, optimized time, gain
5. Computes sum-of-parts vs total (reveals interaction effects)
6. Checks for dominance (no single part > 25% of total)
7. Outputs results to `docs/sensitivity-baseline.md` (machine-readable)

The "optimized" variants must be genuine improvements (not worse than default). Each should represent a smarter-than-default strategy for that part.

The script must be deterministic (same seed → same results). Lock the seed at 42.

This exact script runs again after Phase 1. Same variants, same seed, same track. The only change is the physics engine underneath.
