# T11.4 — Simulation Integration: ERS + Brake Temp Wiring

## Objective

Wire ERS and brake temperature into simulation.py. After T11.1 extraction, simulation.py should be ≤370 lines with room for ~25 lines of new wiring.

## CONSTRAINT

simulation.py must stay ≤ 400 lines, 15 functions after wiring.

## Implementation Plan

### Imports

```python
from .ers_model import create_ers_state, update_ers, get_ers_speed_bonus, reset_ers_lap
from .brake_model import create_brake_state, update_brake_temp, get_brake_efficiency
```

### __init__ — add to car state

```python
"ers": create_ers_state(),
"ers_deploy_mode": "balanced",
"brake_state": create_brake_state(),
```

### _step_car — parse ERS deploy mode from strategy decision

```python
ers_mode = decision.get("ers_deploy_mode", "balanced")
if ers_mode not in ("attack", "balanced", "harvest"):
    ers_mode = "balanced"
state["ers_deploy_mode"] = ers_mode
```

### _apply_physics — ERS speed bonus + brake efficiency

After target speed calculation:
```python
target += get_ers_speed_bonus(state["ers"], state["ers_deploy_mode"])
```

For braking, multiply effective brakes by brake efficiency:
```python
brake_eff = get_brake_efficiency(state["brake_state"]["temp"])
brakes = setup.get("effective_brakes", state["brakes"]) * brake_eff
```

### _step_car — update ERS and brake temp each tick

After physics:
```python
# ERS update — deploy and harvest
braking = max(0, target_speed - state["speed"])  # proxy for braking force
state["ers"] = update_ers(state["ers"], state["ers_deploy_mode"], braking, dt)

# Brake temp update
state["brake_state"] = update_brake_temp(state["brake_state"], braking, state["speed"], dt)
```

On lap change (already detected for tire_age_laps):
```python
state["ers"] = reset_ers_lap(state["ers"])
```

### build_strategy_state — expose new fields

```python
"ers_energy": state["ers"]["energy"],
"ers_deploy_mode": state["ers_deploy_mode"],
"brake_temp": state["brake_state"]["temp"],
```

### Replay — add fields

In replay.py record_frame():
```python
"ers_energy": round(state.get("ers", {}).get("energy", 4.0), 2),
"brake_temp": round(state.get("brake_state", {}).get("temp", 20.0), 1),
```
