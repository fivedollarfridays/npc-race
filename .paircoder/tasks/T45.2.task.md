# T45.2 — Coaching Engine
Create `engine/coaching.py` — generates actionable tips from efficiency data.

## Tips to generate
- Gearbox < 0.95: "Shift at 10800-12500 RPM (peak torque), not {actual_rpm}"
- Cooling < 0.90: "Reduce cooling effort (less drag). Try 0.2-0.4 range"
- Efficiency product < 0.85: "Your car is losing {pct}% to suboptimal parts"
- All parts good: "Excellent! Try a different track or move to Ghost mode"

- [ ] generate_coaching(trial_result) returns list of tip strings
- [ ] Tips reference actual RPM values and efficiency scores
