# T34.3 — Fast Mode Results Export + CLI

Fast mode writes `results.json` + `lap_summary.json` only (no replay.json).
Wire into CLI: fast mode is default, `--live` enables full replay.

## CLI
```
npcrace run --car-dir cars --track monza           # fast (default)
npcrace run --car-dir cars --track monza --live     # full replay
```
