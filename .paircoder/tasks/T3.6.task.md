# Objective

Wire tire_model, fuel_model, pit_lane, lateral, and time calibration into the simulation loop.

# Implementation Plan

1. __init__: accept real_length_m, compute world_scale. Add per-car state: tire_compound="medium", fuel_kg, pit_state, engine_mode="standard", tire_age_laps=0
2. _update_distance: `distance += speed * (1/3.6) * world_scale * dt`
3. Replace _apply_tire_wear body: call tire_model.compute_wear() and compute_grip_multiplier()
4. Add _apply_fuel(): call fuel_model.compute_fuel_consumption(), update fuel_kg
5. _apply_physics: add fuel_model.compute_weight_from_fuel() to mass_factor, use engine_mode power_mult
6. _step_car: extract engine_mode, pit_request, tire_compound_request from decision. Call pit_lane.update_pit_state(). If in pit, apply speed limit. On pit complete, swap compound + reset wear.
7. build_strategy_state: add fuel_remaining, fuel_pct, tire_compound, tire_age_laps, gap_ahead_s, gap_behind_s, engine_mode, pit_window_open
8. race_runner.py: pass real_length_m from track data
9. Backward compat: tire_mode maps to compound when tire_compound not set by strategy

# Acceptance Criteria

- [ ] Full race runs with compounds, fuel, pit stops, lateral, and calibrated timing
- [ ] Old strategies (throttle/boost/tire_mode only) still work
- [ ] simulation.py ≤400 lines, ≤15 functions
- [ ] Procedural tracks work (world_scale=1.0)
- [ ] Named track lap times are realistic (Monaco ~75s)
- [ ] All pre-existing tests pass
