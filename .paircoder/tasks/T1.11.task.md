# Objective

Write integration tests that verify the full pipeline: load cars, select track, run race, produce valid replay JSON. Covers the complete flow from play.py through engine to output.

# Implementation Plan

1. Create `tests/test_integration.py`
2. Test scenarios:
   - Run race with named track, verify replay JSON has correct track_name
   - Run race with procedural track, verify backward compat
