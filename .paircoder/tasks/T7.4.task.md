# T7.4 — Fuel & Pit Calibration from TUMFTM Data

## Objective

Calibrate fuel consumption and pit stop timing using real F1 data from TUMFTM's research. Our fuel consumption is 18% low and pit stops are slightly high.

## Research Data (TUMFTM, Monza 2017)

- Fuel consumption: **1.981 kg/lap** (uniform across teams)
- Starting fuel: **105.0 kg**
- Fuel weight → lap time: **0.027 s/kg** per lap
- Pit standstill (best team): **2.334 s** (Mercedes)
- Pit standstill (worst team): **3.274 s** (Sauber)
- Total pit loss: **22.9-23.9 s** (normal conditions)

## Implementation Plan

### Update `engine/fuel_model.py`

```python
# Current: BASE_CONSUMPTION_KG_PER_M = 0.00028 (gives ~1.62 kg/lap at Monza)
# Target: ~2.0 kg/lap at Monza (5793m) → 0.000345 kg/m
BASE_CONSUMPTION_KG_PER_M = 0.000345

# Verify: 5793 * 0.000345 = ~2.0 kg/lap ✓
```

Also add lap-time sensitivity constant for future use:
```python
FUEL_LAP_TIME_SENSITIVITY = 0.030  # seconds per kg per lap (TUMFTM: 0.027, real data: 0.03-0.04)
```

### Update `engine/pit_lane.py`

```python
# Current: PIT_STATIONARY_TICKS = 660 (22.0s)
# Target: ~20.0s stationary (closer to real 2.3s tire change + 17.8s outlap delta)
PIT_STATIONARY_TICKS = 540  # 18.0s

# Total: 2.0 (entry) + 18.0 (stationary) + 2.0 (exit) = 22.0s
# This matches TUMFTM's 22.9s average well
```

Optionally: reduce further to match real sub-3s stationary times. But the "stationary" phase in our model includes the outlap speed delta (not just tire change), so 18s is reasonable for the combined effect.
