# T34.7 — Qualifying Simulation

Single flying lap qualifying. Each car runs solo: 1 out-lap + 1 flying lap.

## Implementation
Create `engine/qualifying.py`:
- `run_qualifying(cars, track, **kwargs) -> list[dict]`
- For each car: single-car RaceSim, laps=2, fast_mode=True
- Lap 1 = out-lap (warmup), Lap 2 = flying lap (timed)
- Sort by ascending flying lap time, assign grid positions
