# T13.2 — Commentary Generator

## Objective

Create `engine/commentary.py` — converts RaceEvents into human-readable commentary strings. Each event type has a template that produces broadcast-style text.

## Design

**Commentary templates:**
```
OVERTAKE:    "LAP {lap} — {car} passes {other} for P{position}!"
BATTLE:      "BATTLE — {car1} vs {car2}, gap {gap}s for {duration} laps"
PIT_STOP:    "PIT STOP — {car} boxes for {compound} tires ({time}s stationary)"
SAFETY_CAR:  "⚠ SAFETY CAR DEPLOYED — {reason}"
SPIN:        "SPIN — {car} loses it at Turn {sector}!"
DNF:         "RETIREMENT — {car} out of the race (damage)"
FASTEST_LAP: "FASTEST LAP — {car} sets {time}s"
```
