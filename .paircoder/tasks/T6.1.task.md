# T6.1 — Extract Physics to engine/physics.py

## Objective

simulation.py is at 387/400 lines and 15/15 functions — at both arch limits. Before we can add timing infrastructure or recalibrate physics, we must extract the speed/acceleration/braking/drafting logic into a standalone `engine/physics.py` module. This is a pure refactor — no behavior changes.

## Implementation Plan

### New file: `engine/physics.py` (~120 LOC)

Move ALL physics constants from simulation.py:
```python
BASE_SPEED = 155
POWER_SPEED_FACTOR = 90
WEIGHT_SPEED_PENALTY = 60
CURVATURE_FACTOR = 47.0
GRIP_BASE_SPEED = 60
GRIP_SPEED_RANGE = 300
ACCEL_BASE = 50
ACCEL_POWER_FACTOR = 60
WEIGHT_MASS_FACTOR = 1.2
BRAKE_BASE = 80
BRAKE_FACTOR = 100
DRAFT_BONUS_BASE = 8
DRAFT_MAX_DISTANCE = 40
```

Extract these pure functions:
```python
def compute_target_speed(power, grip, weight, aero, curvature, throttle,
                          setup=None) -> float:
    """Calculate target speed for current conditions."""

def compute_acceleration(speed, target_speed, power, weight, fuel_weight, dt) -> float:
    """Calculate new speed after acceleration toward target."""

def compute_braking(speed, target_speed, brakes, dt) -> float:
    """Calculate new speed after braking toward target."""

def compute_draft_bonus(aero, distance_ahead, dt) -> float:
    """Calculate speed bonus from drafting car ahead."""

def compute_mass_factor(weight, fuel_weight) -> float:
    """Calculate mass factor from weight stat + fuel."""
```

### Update `engine/simulation.py`

- Remove all 13 physics constants (moved to physics.py)
- Remove inline physics math from `_apply_physics()` — replace with function calls
- `_apply_physics()` becomes an orchestrator: reads state, calls physics functions, writes results
- Target: simulation.py drops from ~387 → ~300 lines, functions stay ≤ 15

### Export via `engine/__init__.py`

```python
from .physics import compute_target_speed, compute_acceleration, compute_braking
```

## Tests

No NEW test file needed — this is a refactor. All existing tests must pass unchanged. Add a small verification:

`tests/test_physics.py` (new, ~30 LOC):
- `test_physics_constants_exist` — import and verify all 13 constants
- `test_compute_target_speed_basic` — known inputs → expected output
- `test_compute_acceleration_bounded` — speed never exceeds target
- `test_compute_braking_bounded` — speed never drops below target

- `engine/physics.py` exists, ≤ 130 lines, ≤ 8 functions
- `simulation.py` drops to ≤ 320 lines (freed ~70 lines of constants + inline math)
