# Objective

Copy `bot_scanner.py` from npc-wars and adapt it for NPC Race car files. The scanner statically analyzes car .py files for dangerous imports, system calls, network access, and file I/O before loading them.

# Implementation Plan

1. Copy npc-wars `bot_scanner.py` as starting point
2. Adapt scan targets for car files:
   - Required exports: CAR_NAME, CAR_COLOR, POWER, GRIP, WEIGHT, AERO, BRAKES
   - Optional: strategy function
   - Blocked: os, sys, subprocess, socket, importlib, __import__, eval, exec, open, pathlib
3. Add car-specific validation (stat budget <= 100, valid hex color, etc.)
4. Place at `security/bot_scanner.py`
5. Write tests

# Acceptance Criteria

- [ ] `scan_car_file(path)` returns ScanResult with passed/failed + reasons
- [ ] Detects dangerous imports (os, sys, subprocess, socket, etc.)
- [ ] Detects dangerous builtins (eval, exec, __import__, open, compile)
- [ ] Validates stat budget <= 100
- [ ] Validates CAR_NAME is non-empty string
- [ ] Validates CAR_COLOR is valid hex
- [ ] Validates all 5 stats present and numeric >= 0
- [ ] All seed cars pass scanning
- [ ] File under 400 lines
- [ ] Tests cover all detection cases
