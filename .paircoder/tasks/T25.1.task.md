# T25.1 — Fix Engine Heat Model

`engine/powertrain_physics.py:81`: change heat coefficient from `15` to `1.2`.

Current: `heat = torque_pct * (rpm / 10000) * 15 * dt` → equilibrium at 410°C.
Fixed: `heat = torque_pct * (rpm / 10000) * 1.2 * dt` → equilibrium at ~110°C.

With combined cooling coeff 0.08:
- cooling=0.1 → ~212°C (overheating)
- cooling=0.3 → ~124°C (borderline)
- cooling=0.5 → ~106°C (good)
- cooling=1.0 → ~93°C (overcooled)

## AC
- Engine temp stabilizes at 100-130°C with cooling=0.5 at moderate load
- Engine overheats (>150°C) with cooling=0.0
- Existing powertrain tests still pass
