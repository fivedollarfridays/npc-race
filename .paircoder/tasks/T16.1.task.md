# T16.1 — Season Calendar

## Objective

Create `engine/season.py` — defines championship season calendars with curated track sequences. Provides preset seasons and custom calendar support.

## Design

**Preset seasons:**
- `"short"`: 5 races — monza, monaco, silverstone, spa, bahrain
- `"full"`: 10 races — monza, monaco, silverstone, spa, bahrain, singapore, interlagos, hungaroring, baku, suzuka
- `"classic"`: 8 races — handpicked for variety (mix of power/technical/street)

**Calendar structure:**
```python
{"name": "2026 Championship", "races": [
    {"round": 1, "track": "bahrain", "laps": 5},
    {"round": 2, "track": "monza", "laps": 5},
    ...
]}
```
