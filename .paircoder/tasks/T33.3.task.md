## GETTING_STARTED.md (repo root)

2. Prerequisites (Python 3.11+, pip install)
3. Quick start: `npcrace init my_car && npcrace run --car-dir my_car --track monza --laps 1`
4. **The before/after moment**: run with default gearbox (shifts at 12800), note lap time.
