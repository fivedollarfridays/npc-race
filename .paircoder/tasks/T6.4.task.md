# T6.4 — Timing Module

## Objective

Create `engine/timing.py` — tracks per-lap times, sector splits, fastest laps, and elapsed race time. Racing without timing is not racing. This module is called by the simulation each tick and produces data for replay enrichment (T6.5).

## Implementation Plan

### Sector definition

Each track is divided into 3 sectors by distance percentage:
- Sector 1: 0%–33% of lap
- Sector 2: 33%–66% of lap
- Sector 3: 66%–100% of lap

Named tracks can optionally override with custom sector boundaries.

### New file: `engine/timing.py` (~100 LOC)

```python
class CarTiming:
    """Per-car timing state."""
    def __init__(self, name: str):
        self.name = name
        self.lap_times: list[float] = []         # seconds per completed lap
        self.sector_times: list[list[float]] = [] # [lap][sector] times
        self.current_lap_start: int = 0          # tick when current lap started
        self.current_sector: int = 0             # 0, 1, 2
        self.current_sector_start: int = 0       # tick when current sector started
        self.best_lap: float | None = None       # best lap time in seconds
        self.best_sectors: list[float | None] = [None, None, None]

SECTOR_BOUNDARIES = (0.333, 0.666, 1.0)  # default 3 equal sectors

def get_sector_boundaries(track_dict: dict) -> tuple[float, float, float]:
    """Return sector boundary percentages. Custom if defined, else equal thirds."""
    return tuple(track_dict.get("sector_boundaries", SECTOR_BOUNDARIES))

def create_timing(car_names: list[str]) -> dict[str, CarTiming]:
    """Initialize timing state for all cars."""

def update_timing(
    timings: dict[str, CarTiming],
    car_name: str,
    distance_pct: float,   # 0.0-1.0 position in current lap
    lap: int,              # current lap number
    tick: int,             # current tick
    ticks_per_sec: int,    # for time conversion
    sector_boundaries: tuple[float, float, float],
) -> dict:
    """
    Update timing for one car. Returns dict with:
    - "lap_completed": bool (just crossed start/finish)
    - "lap_time": float|None (if lap just completed, in seconds)
    - "sector_completed": bool
    - "sector_time": float|None (if sector just completed)
    - "current_sector": int (0, 1, 2)
    - "elapsed_s": float (total race time for this car)
    """

def get_fastest_lap(timings: dict[str, CarTiming]) -> tuple[str, float] | None:
    """Return (car_name, best_time) for the fastest lap across all cars, or None."""

def get_timing_summary(timings: dict[str, CarTiming]) -> list[dict]:
    """Return per-car timing summary for results enrichment."""
```

### Integration hooks

simulation.py will call `update_timing()` per car per tick. The timing data will be stored alongside car state but NOT in the car state dict (separate `self.timings` dict on RaceSim).

### Track sector data

Add optional `sector_boundaries` to 3-4 named tracks:
- Monza: `(0.35, 0.65, 1.0)` — long straight dominates S1
- Monaco: `(0.30, 0.65, 1.0)` — slow S2 through swimming pool

Others default to equal thirds.
