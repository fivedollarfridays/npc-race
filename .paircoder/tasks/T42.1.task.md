# T42.1 — Extend PartsRaceSim for Race Runner Compatibility

PartsRaceSim is missing parameters that run_race() passes:
- `fast_mode` (1fps frame storage + LapAccumulator)
- `drs_zones` (DRS zone detection)
- `car_data_dir`, `race_number` (can be stored but unused)

## Implementation
Add these params to PartsRaceSim.__init__. For fast_mode, replicate
the same pattern from RaceSim: record frames every TICKS_PER_SEC
ticks instead of every tick, integrate LapAccumulator.

For drs_zones, store but don't implement DRS physics yet (mark TODO).
