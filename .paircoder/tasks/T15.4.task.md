# T15.4 — Wire Director into main.js

## Objective

Wire the TV Director into the main playback loop. Add a "Director" camera mode button. Call updateDirector each frame with events.

## Changes to `viewer/js/main.js`

1. Add "Director" to camera mode buttons (alongside full/follow/onboard)
2. In tick(), when mode is "director":
   ```javascript
   if (cameraSystem.mode === 'director') {
     updateDirector(replay, frame, replay.events || []);
     var target = getDirectorTarget();
     // Set camera to follow target car
   }
   ```
3. Call `drawSectorMarkers` and `drawDRSZones` in render pipeline
