### Gap compression under SC

In `step()`, after SC update:
```python
if should_compress_gaps(self.safety_car):
    # Gradually compress gaps toward 1.0s
    by_dist = sorted(self.states, key=lambda s: -s["distance"])
    for i in range(1, len(by_dist)):
        if by_dist[i]["finished"]:
            continue
        leader_dist = by_dist[i-1]["distance"]
        target_gap = 1.0 * by_dist[i-1]["speed"] * (1/3.6) * self.world_scale / self.TICKS_PER_SEC
        actual_gap = leader_dist - by_dist[i]["distance"]
        if actual_gap > target_gap * 2:
            by_dist[i]["distance"] += (actual_gap - target_gap) * 0.01  # gradual compression
```
