# T36.6 — Verify CI Timing (Integration Gate)

Push the changes, run CI, verify timing targets.

## Targets
- Unit tests: < 1 min
- Smoke tests: < 3 min
- Total on-push CI: < 3 min (parallel jobs)
- Integration tests: < 15 min (on demand)

## Verification steps
1. Push to branch
2. Watch CI run
3. Record job durations
4. If any job exceeds target, identify slow tests and fix
5. Run integration suite locally, verify < 15 min

## AC
- [ ] CI on-push completes in < 3 min
- [ ] No test timeout failures
- [ ] All 2,310+ tests still pass across all tiers
- [ ] Zero test regressions
