# T33.1 — Leaderboard Module

Create `engine/leaderboard.py` with persistent local standings.

## Functions
- load_leaderboard(path) -> dict: load from JSON file (create empty if missing)
- add_result(leaderboard, results_summary) -> dict: update standings with race results
- save_leaderboard(leaderboard, path) -> None: persist to JSON
- format_standings(leaderboard) -> str: printable standings table

## Standings Format
```json
{
  "version": "1.0",
  "last_updated": "2026-03-23T12:00:00Z",
  "entries": [{
    "name": "CarName",
    "league": "F3",
    "races": 5,
    "wins": 2,
    "podiums": 4,
    "best_lap_s": 80.12,
    "avg_position": 1.8,
    "total_points": 95,
    "reliability_score": 0.94
  }]
}
```

## Points: F1 championship (25, 18, 15, 12, 10, 8, 6, 4, 2, 1) for P1-P10.
