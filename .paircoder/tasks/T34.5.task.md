# T34.5 — Spatial Neighbor Lookup

Replace O(n^2) drafting/lateral loops with sorted-position O(1) lookup.

## Implementation
1. Create `engine/spatial.py` with `SortedCarIndex`
2. Sort states by distance once per tick
3. `neighbors(car_idx, max_dist)` uses bisect
4. Replace inner loops in sim_physics.py `apply_drafting`/`apply_lateral`
