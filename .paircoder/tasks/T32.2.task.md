# T32.2 — Integrity Hash

Add SHA-256 hash to results summary to detect tampering.

## Implementation
- engine/results.py: `compute_integrity_hash(results_dict)` -> str
- Hash computed over: car results + track + laps + league (NOT timestamp, NOT the hash itself)
- Stored as `results["integrity"] = "sha256:hexdigest"`
- Verification: `verify_integrity(results_dict)` -> bool

## Why
Players run races locally and submit results. Without integrity checking,
someone could edit the JSON to change their lap time. The hash makes
tampering detectable (not impossible, but detectable).

## AC
- Hash computed and included in results summary
- Modifying any result field breaks the hash
- verify_integrity returns True for unmodified, False for tampered
