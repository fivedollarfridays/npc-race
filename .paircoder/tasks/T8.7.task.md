# T8.7 — Integration Gate: Dashboard Verification

## Objective

Verify the entire pit wall dashboard works end-to-end: `play.py --track monza` opens dashboard.html, all 4 zones render with live data, car selection works, diagnostic mode works at race end. Python test suite still passes.

## Verification Checklist

### Python side
1. `python -m pytest --ignore=tests/test_balance_v2.py` — all tests pass (≥ 1113)
2. `ruff check .` — clean
3. Replay frames contain all new fields (gap_behind_s, last_lap_time, best_lap_s, tire_age_laps, pit_stops, dirty_air_factor)
4. `play.py` opens dashboard.html by default

### Dashboard HTML
1. `viewer/dashboard.html` exists and loads
2. All panel IDs present: statusBar, timingTower, trackView, carTelemetry, telemetryStrip
3. Script tags for all JS modules (existing 8 + 4 new dashboard modules)
4. CSS grid layout renders correctly at 1920x1080 and 1440x900

### Timing Tower
1. 5 car rows visible with position, abbreviation, gap, compound, wear bar
2. Click row selects car (fires car-selected event)
3. Fastest lap marked purple
4. Cars in pit show red tint
5. Finished cars show "FIN"

### Car Telemetry Panel
1. 10 readouts update live with correct values and colors
2. Sector comparison table shows current vs best with deltas
3. Dirty air alert fires when entering dirty air
4. Tire cliff warning fires near cliff threshold
5. Undercut/overcut alert fires on pit-related position changes

### Telemetry Strip
1. Speed trace renders with curvature background
2. Tire wear + temp chart shows dual axes
3. Gap chart shows green/red fill for gain/loss
4. Rolling 3-lap window scrolls
5. Pit stop markers visible

### Diagnostic Mode
1. "DIAGNOSTIC" button appears at race end
2. Lap time bar chart renders with compound colors
3. Strategy timeline shows stints
4. Sector table has purple/green/yellow coloring
5. Only available for player's car

### Smoke Tests
```bash
python play.py --track monza --laps 5 --no-browser
# Then open http://localhost:8765/dashboard.html manually
# Verify all panels render and update
```
