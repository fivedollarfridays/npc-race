# T19.1 — Track Speed Profile

## Objective

Create `engine/speed_profile.py` — pre-computes the optimal speed at every track point for a given car. This is the "ideal lap" that a perfect driver would follow. Accounts for braking distance — you can't go 300 at a point if the next point requires 100 and you can't brake in time.

## Design

**Forward pass:** Compute max cornering speed at each point from curvature + grip.

**Backward pass:** Walk backwards through the profile. If point N requires speed S, and point N-1 has speed S', then S' can't exceed what's achievable by braking from S' to S over the distance between points. This creates realistic braking zones.

**Forward acceleration pass:** Walk forwards. If point N has speed S and point N+1 allows speed S', then S' can't exceed S + max_acceleration. This creates realistic acceleration zones.

The result: a smooth speed curve that goes high on straights and low in corners, with realistic braking and acceleration ramps.
