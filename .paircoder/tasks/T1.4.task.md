# Objective

Copy `sandbox.py` from npc-wars and adapt for NPC Race. The sandbox wraps car strategy() calls with timeout protection and exception handling, ensuring a bad car file cannot crash the race.

# Implementation Plan

1. Copy npc-wars `sandbox.py` as starting point
2. Adapt for car strategy execution:
   - Wrap `strategy(state)` calls with timeout (100ms per call)
   - Catch all exceptions, return default decisions on failure
   - Default decisions: throttle=1.0, boost=False, tire_mode="balanced"
   - Log warnings for misbehaving cars (don't crash race)
3. Place at `security/sandbox.py`
4. Integrate with engine car loader
5. Write tests

# Acceptance Criteria

- [ ] `safe_strategy_call(strategy_fn, state, timeout_ms)` returns decisions dict
- [ ] Returns default decisions on exception
- [ ] Returns default decisions on timeout
- [ ] Returns default decisions on bad return type
- [ ] Merges partial returns with defaults (missing keys filled in)
- [ ] Validates throttle is float 0.0-1.0, clamps if needed
- [ ] Validates tire_mode is valid enum value
- [ ] File under 400 lines
- [ ] Tests cover exception, timeout, bad return, partial return cases
