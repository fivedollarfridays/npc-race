# T47.2 — Tiered Race Command
`npcrace run` defaults to player's current tier. `--full-grid` for all 20.

## Implementation
- Read progression state from `~/.npcrace/progress.json`
- Default tier: "rookie" for new players
- `--tier rookie|midfield|front|full` flag
- `--full-grid` shortcut for --tier full
