# T14.4 — Camera-Aware Mix + Pit Wall Audio

## Objective

Adjust audio mix based on camera mode. Onboard = engine dominant. Follow = balanced. Full track = ambient-heavy. Pit wall = louder when near pit lane.

## Design

**Camera modes (from viewer/js/camera.js):**
- `full` — full track view: engine quiet, crowd loud, ambient dominant
- `follow` — follow selected car: engine medium, balanced mix
- `onboard` — cockpit view: engine loud, wind noise, tire squeal prominent

**Mix profiles:**
```
            engine  crowd  aero   tire   turbo
full:       0.2     0.8    0.1    0.1    0.1
follow:     0.6     0.4    0.3    0.2    0.3
onboard:    1.0     0.1    0.5    0.4    0.5
```

**Pit wall:** When camera is near pit entry/exit zones, add a subtle pit lane ambience (hydraulic hiss = filtered noise).

## Changes to `viewer/js/sound-engine.js`

1. Add `setCameraMix(mode)` — adjusts gain levels per camera mode
2. Add `_mixProfiles` object with gain multipliers
3. Wire from `main.js` when camera mode changes

## Tests

- `test_camera_mix_function` — JS has `setCameraMix`
- `test_mix_profiles_exist` — JS has `_mixProfiles` object
- `test_camera_modes_covered` — profiles for full/follow/onboard
