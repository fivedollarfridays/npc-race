# T36.3 — Shared Test Fixtures for Sim Data

Many tests run the sim just to get result data they could mock.
Create shared fixtures that provide pre-built race results, replay
data, and lap summaries without running the simulation.

## Create tests/fixtures/

### tests/fixtures/__init__.py
### tests/fixtures/race_data.py
- `SAMPLE_RESULTS` — list of 6 car result dicts (position, time, laps, etc.)
- `SAMPLE_LAP_SUMMARIES` — per-car per-lap data for 3 laps
- `SAMPLE_REPLAY_METADATA` — minimal replay dict structure
- `SAMPLE_GRID` — qualifying grid for 6 cars
- `make_results(n_cars=6, n_laps=3)` — factory for custom sizes

### tests/conftest.py updates
- `@pytest.fixture` for `sample_results`, `sample_lap_summaries`, etc.
- Available to all test files automatically
