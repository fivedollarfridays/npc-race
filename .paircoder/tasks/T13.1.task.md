# T13.1 — Event Detection

## Objective

Create `engine/narrative.py` — scans replay frames to detect race events: battles, overtakes, pit stops, safety car deployments, spins, DNFs, fastest laps.

## Design

**Event types:**
- `BATTLE`: two cars within 1.0s gap for 90+ consecutive frames (3s)
- `OVERTAKE`: position change between two nearby cars
- `PIT_STOP`: car enters pit (pit_status changes from "racing")
- `SAFETY_CAR`: safety_car field changes to True
- `SPIN`: in_spin field changes to True
- `DNF`: car finishes with damage DNF
- `FASTEST_LAP`: new best lap time set

Each event has: type, tick, car_name(s), description data.
