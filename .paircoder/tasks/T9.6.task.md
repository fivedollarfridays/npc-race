# T9.6 — Seed Cars + Strategy State + Replay Enrichment

## Objective

Update car_template.py with documentation for new fields. Update 2-3 seed cars to react to damage/SC/spin risk. Add new fields to replay frames.

## car_template.py documentation

Add:
```python
# Sprint 9: Collisions & Safety Car
# state["damage"]          float 0.0-1.0 — accumulated damage (reduces aero, grip, speed)
# state["spin_risk"]       float 0.0-1.0 — current probability of spinning this tick
# state["safety_car"]      bool — safety car currently active
# state["safety_car_laps"] int — laps of SC remaining (0 if inactive)
# state["in_spin"]         bool — car currently recovering from a spin
# state["contact_last_lap"] bool — had contact this lap

## Seed car updates (2-3 cars)

**GooseLoose** — SC-aware pitting:
- If `safety_car` is True and `pit_stops == 0` and `tire_wear > 0.3`: pit NOW (free pit window)

**SlipStream** — damage-aware:
- If `damage > 0.3`: switch to conserve mode (reduce spin risk)

**BrickHouse** — aggression parameter:
- Return new `"aggression": 0.3` when `spin_risk > 0.001` (back off when risky)
- Return `"aggression": 0.7` when `safety_car` and gap is close (attack on restart)

## Replay enrichment

```
