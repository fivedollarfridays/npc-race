# T5.1 — Tire Temperature Model

## Objective

Create `engine/tire_temperature.py` — a standalone functional module that models tire thermal state. Tires have an optimal operating window; outside it, grip degrades. Cold tires (out-lap) give less grip; overheated tires blister and lose grip faster.

## Implementation Plan

### New file: `engine/tire_temperature.py` (~100 LOC)

Constants:
```python
OPTIMAL_TEMP = {"soft": 90.0, "medium": 80.0, "hard": 70.0}   # °C peak grip
TEMP_WINDOW  = {"soft": 20.0, "medium": 25.0, "hard": 30.0}   # ±°C around optimal
AMBIENT_TEMP = 20.0   # baseline dissipation target
```

Functions (all pure, no side effects):
- `heat_generation(throttle, curvature, lateral, dt) -> float` — heat generated this tick (°C/tick)
  - `throttle * 0.4 + curvature * 1.5 + abs(lateral) * 0.3` scaled by dt
- `heat_dissipation(tire_temp, speed, dt) -> float` — passive + airflow cooling
  - `((tire_temp - AMBIENT_TEMP) * 0.015 + speed * 0.002) * dt`
- `update_tire_temp(tire_temp, heat_gen, heat_diss) -> float` — clamp to [AMBIENT_TEMP, 150.0]
- `tire_temp_grip_factor(tire_temp, compound) -> float` — 0.5–1.0 multiplier
  - Below window: linear from 0.5 (cold) to 1.0 (optimal low edge)
  - In window: 1.0
  - Above window: linear from 1.0 to 0.5 (blistered), capped at 150°C

### Integration hooks (no sim changes yet — T5.4 wires this in)

Export via `engine/__init__.py`:
```python
from .tire_temperature import heat_generation, heat_dissipation, update_tire_temp, tire_temp_grip_factor
```

## Tests: `tests/test_tire_temperature.py`

- `test_heat_generation_increases_with_throttle` — full throttle > half throttle
- `test_heat_generation_increases_with_curvature` — corner generates more heat than straight
- `test_heat_dissipation_increases_with_speed` — faster speed → more cooling
- `test_update_tire_temp_clamps_to_min` — never below AMBIENT_TEMP
- `test_update_tire_temp_clamps_to_max` — never above 150°C
- `test_grip_factor_at_optimal_temp` — returns 1.0 inside window
- `test_grip_factor_cold_tire` — < 1.0 below window, monotonically increasing
- `test_grip_factor_blistered_tire` — < 1.0 above window, monotonically decreasing
- `test_compound_specific_windows` — soft optimal != hard optimal
- `test_tire_warms_from_cold_under_load` — 20°C rises toward 90°C over many ticks
