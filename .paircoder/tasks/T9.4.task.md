# T9.4 — Safety Car System

## Objective

Create `engine/safety_car.py` — a state machine for the safety car. When triggered by an incident, all cars slow to SC pace, gaps compress, pit window opens with reduced time loss. Lasts 3-5 laps. Restart procedure with 1-lap warning.

## Design

**Safety Car States:**
```
INACTIVE → DEPLOYED → ACTIVE → ENDING → INACTIVE
```

- `INACTIVE`: Normal racing
- `DEPLOYED`: SC comes out, cars slow down (1 lap transition)
- `ACTIVE`: Cars bunched at SC pace, gaps compressed to ~1.0s
- `ENDING`: "SAFETY CAR IN THIS LAP" — 1 lap warning before restart
- `INACTIVE`: Green flag, racing resumes

**Effects during SC:**
- All cars speed-limited to SC_PACE (120 km/h)
- Gaps between cars compressed toward 1.0s over 2 laps
- Tire degradation reduced to 25% (tires cool, less stress)
- Fuel consumption reduced to 25%
- Pit stop time loss reduced by ~8s (slower pit-in/out delta)
- Lapped cars may unlap themselves (simplified: just compress gaps)

**Triggers:**
- DNF from collision (100% SC)
- Spin in high-curvature zone (20% SC)
- Multi-car contact (40% SC)
- Called by `trigger_safety_car(reason, rng)` from simulation

**Duration:** 3-5 laps (random, weighted by severity)

## New file: `engine/safety_car.py` (~120 LOC)

```python
SC_PACE = 120.0              # km/h under safety car
SC_MIN_LAPS = 3
SC_MAX_LAPS = 5
SC_GAP_TARGET = 1.0          # seconds — gaps compress to this
SC_TIRE_DEG_MULT = 0.25      # tire wear rate during SC
SC_FUEL_MULT = 0.25          # fuel consumption during SC
SC_PIT_TIME_REDUCTION = 240  # ticks saved on pit stop under SC (~8s)

class SafetyCarState:
    """Safety car state machine."""
    INACTIVE = "inactive"
    DEPLOYED = "deployed"
    ACTIVE = "active"
    ENDING = "ending"

def create_safety_car_state() -> dict:
    """Return initial SC state."""
    return {
        "status": SafetyCarState.INACTIVE,
        "laps_remaining": 0,
        "laps_active": 0,
        "reason": None,
        "deploy_tick": None,
    }

def trigger_safety_car(sc_state: dict, reason: str, rng, tick: int) -> dict:
    """Deploy safety car. Returns updated state."""

def update_safety_car(sc_state: dict, leader_lap: int, tick: int) -> dict:
    """Advance SC state machine. Called each tick."""

def get_sc_speed_limit(sc_state: dict) -> float | None:
    """Return speed limit under SC, or None if no SC."""

def get_sc_modifiers(sc_state: dict) -> dict:
    """
    Return modifier dict for tire/fuel/pit when SC active:
    {"tire_deg_mult": 0.25, "fuel_mult": 0.25, "pit_time_reduction": 240}
    or {"tire_deg_mult": 1.0, "fuel_mult": 1.0, "pit_time_reduction": 0}
    """

def is_sc_active(sc_state: dict) -> bool:
    """True if SC is deployed, active, or ending."""

def should_compress_gaps(sc_state: dict) -> bool:
    """True if gaps should be compressed toward SC_GAP_TARGET."""
```
