# T25.5 — Grip Factor Replaces Profile Hack

Replace `profile_speed *= (0.30 + 0.70 * prev_eff)` with `profile_speed *= grip_factor`.

grip_factor = effective_mu / profile_mu where:
- effective_mu = tire_mu * (weight + downforce) / weight * diff_traction_mult
- profile_mu = 4.0 (from speed_profile.py: 3.2 + grip * 1.6 at grip=0.5)

## Changes
- Add compute_grip_factor() to efficiency_engine.py
- Store grip_factor on car state
- parts_simulation.py: replace profile hack with grip_factor lookup

## AC
- No 0.30 + 0.70 * prev_eff in parts_simulation.py
- Good suspension → faster corners (measurable)
- Worn tires → slower corners (measurable)
