# Objective

Upgrade all 5 seed cars to use load_data/save_data for cross-race learning.

# Implementation Plan

- BrickHouse: records which pit lap gave best position, adjusts pit window over races
- GlassCanon: records 0-stop vs 1-stop result per track, switches strategy accordingly

- GooseLoose: profiles opponent speeds, adjusts undercut/overcut timing
- SlipStream: records drafting effectiveness, adjusts lateral positioning
