# T8.2 — Dashboard HTML/CSS Layout

## Objective

Replace the current single-canvas viewer with a 4-zone CSS grid layout. No JS logic yet — just the structural HTML, CSS dark theme, and placeholder panels. The existing track view becomes one panel in the center.

## Layout Structure

```
┌──────────────────────────────────────────────────────────────┐
│ STATUS BAR (full width, 36px)                                │
├──────────┬──────────────────────┬────────────────────────────┤
│ TIMING   │ TRACK VIEW           │ CAR TELEMETRY              │
│ TOWER    │ (existing canvases)  │ PANEL                      │
│ (200px)  │ (flex: 1)            │ (340px)                    │
│          │                      │                            │
├──────────┴──────────────────────┴────────────────────────────┤
│ TELEMETRY STRIP (full width, 200px)                          │
├──────────────────────────────────────────────────────────────┤
│ CONTROLS (full width, 48px — existing play/pause/scrubber)   │
└──────────────────────────────────────────────────────────────┘
```

## New File: `viewer/dashboard.html`

Keep `viewer.html` as legacy. Create `viewer/dashboard.html` as the new default.
