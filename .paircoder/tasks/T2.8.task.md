# Objective

Add physics-based visual effects: tire marks on track, enhanced brake glow, drafting wake particles, and visual wheel spin.

# Implementation Plan

1. Create `viewer/js/physics-fx.js` with effect functions
2. Tire marks: at track points where curvature > threshold AND car speed > threshold, draw short dark lines on the background canvas at car position. Marks persist across frames (accumulate over race). Reset on replay restart.
3. Enhanced brake glow: brief red flash trail behind car on hard braking (speed drop > threshold). Drawn on car layer behind the car.
4. Drafting wake: when car is within 5-40 world units behind another car, draw faint translucent lines/particles between them showing slipstream effect
5. Visual wheel spin: wheel texture rotation angle increments proportional to car speed

# Acceptance Criteria

- [ ] Tire marks visible on track surface after cars pass through corners
- [ ] Tire marks persist across frames but reset on replay restart/scrub
- [ ] Brake glow visible behind cars during hard braking
- [ ] Drafting effect visible between closely spaced cars
- [ ] Effects do not impact frame rate (< 2ms per frame for all effects combined)

# Verification

- Build viewer, play race, verify tire marks accumulate in corners
- Restart replay — verify marks reset
- Watch close racing — verify drafting wake between cars
