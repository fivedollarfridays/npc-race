# T16.4 — Season Runner

## Objective

Create `engine/season_runner.py` — orchestrates a full championship season: iterates through the calendar, runs each race, awards points, applies development, tracks standings.

## Design

**Season flow:**
1. Load season calendar (from T16.1)
2. Load cars
3. For each race in calendar:
   a. Run race via `run_race()`
   b. Award championship points (T16.2)
   c. Award development points (T16.3)
   d. Print race results + standings after each round
4. Print final championship standings
5. Return season results dict

**Output:** season results JSON with per-race results, final standings, development history.
