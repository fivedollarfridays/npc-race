# T30.2 — Quality Gate Enforcement

Add advisory and enforced quality gates to league_system.py.

## Advisory (F3/F2)
- Compute quality metrics (CC, lint, type hints)
- Generate report showing cost: "CC 8 would cost 0.5s/lap at F1 level"
- Never block entry — always allow racing

## Enforced (F1/Championship)
- F1: ruff clean required (0 lint violations)
- Championship: ruff clean + CC < 15 per function + reliability >= 0.88
- Rejection with specific feedback: "gearbox.py CC=18 (limit 15)"

## Functions
- generate_quality_report(car, league) → QualityReport
- QualityReport: passed, advisory_messages, blocking_violations, reliability_score

## AC
- F3 car with lint violations: advisory report, allowed to race
- F1 car with lint violations: rejected with specific violation list
- Championship car with CC > 15: rejected with function-level feedback
