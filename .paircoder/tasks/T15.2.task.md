# T15.2 — 3-Letter Labels + Car Color Matching

## Objective

Replace full car names with 3-letter abbreviations in the viewer. Match label colors to car colors for instant visual identification.

## Design

**Abbreviation:** First 3 characters uppercase: "GooseLoose" → "GOO", "BrickHouse" → "BRI", "SlipStream" → "SLI", "Silky" → "SIL", "GlassCanon" → "GLA"

**Color matching:** Car labels on the track renderer use the car's CAR_COLOR instead of white text.

## Changes

### `viewer/js/car-renderer.js`
- Add `getCarAbbrev(name)` — returns first 3 chars uppercase
- Modify car label rendering to use abbreviation
- Set label fill color to car's color field

### `viewer/js/timing-tower.js`
- Tower rows already show 3-char names (line ~30 in initTimingTower)
- Verify color is applied from car color

## Tests

- `test_car_renderer_has_abbrev_function` — car-renderer.js has `getCarAbbrev`
- `test_abbrev_logic_in_renderer` — renderer references abbreviation
- `test_car_color_in_labels` — renderer uses car color for labels

## Acceptance Criteria

- Car labels show 3-letter abbreviations
- Labels colored to match car color
- 3 new tests pass
