# Objective

Rewrite all 5 seed cars with reactive adaptation — decisions based on tire_wear, gaps, position, and curvature instead of fixed lap percentages.

# Implementation Plan

- BrickHouse: pit when tire_wear>0.7 AND gap_ahead_s>25 (not fixed 25%/60%), emergency pit at 0.85
- GlassCanon: emergency pit if tire_wear>0.85 (grip cliff protection), adaptive tire_mode
- GooseLoose: position+gap-aware engine mode (conserve when leading by >3s, push when chasing within 2s)
- Silky: adaptive lateral (inside in corners, block on straights when gap_behind_s<1.0)
- SlipStream: gap-aware pit timing, pit when gap allows recovery

# Acceptance Criteria

- [x] BrickHouse no longer finishes last in every race
