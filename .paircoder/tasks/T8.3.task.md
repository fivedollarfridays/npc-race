# T8.3 — Timing Tower JS

## Objective

Build the left-column timing tower: a live leaderboard that updates every frame. Shows position, car name abbreviation, gap to leader, tire compound + age, and tire wear bar. Clicking a row selects that car for the telemetry panel.

## New File: `viewer/js/timing-tower.js` (~150 LOC)

### Data per row

```
P1  GOO  +0.000s   M  12  ██████░░ 62%
```

Each row (50px height) contains:
- Position number (bold, 24px)
- Car color stripe (4px left border)
- Car abbreviation (3-char: first 3 of name, uppercased)
- Gap to leader (+X.XXXs, or "LEADER" for P1)
- Tire compound dot (red=S, yellow=M, white=H)
- Tire age (laps)
- Tire wear mini-bar (colored: green→yellow→red by wear level)

### Selected car highlight

- Default: player's car (first car in replay, or configurable)
- Click row: select that car → highlighted background (`--bg-panel-alt`)
- Selected car dispatches a custom event: `document.dispatchEvent(new CustomEvent('car-selected', {detail: {name: carName}}))`

### Fastest lap indicator

- Purple dot (●) next to the car that holds the overall fastest lap
- Updated each frame by comparing `best_lap_s` across all cars

### Functions

```javascript
function initTimingTower(replay) { /* Create row DOM elements */ }
function updateTimingTower(frame, selectedCar) { /* Update all rows from frame data */ }
function selectCar(carName) { /* Highlight row, dispatch event */ }
```

### Color coding

- Gap gaining (vs previous frame): white text
- Gap losing: yellow text
- Car in pit: red background tint on row
- Finished car: gray text, "FIN" instead of gap

## Integration

`main.js` calls `initTimingTower(replay)` on load and `updateTimingTower(frame, selectedCar)` on each tick.

## Acceptance Criteria

- `viewer/js/timing-tower.js` exists, ≤ 180 LOC
- 5 car rows render with correct data
- Click selects car, fires custom event
- Fastest lap marked with purple dot
- Tire compound colors match F1 convention
- Works with dashboard.html layout
