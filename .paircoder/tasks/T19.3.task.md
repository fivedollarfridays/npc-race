# T19.3 — Deterministic Driver

## Objective

Create `engine/driver_model.py` — the F1 driver AI. Takes current car state + track data, returns optimal driving inputs (throttle, lateral_target). Uses the speed profile and racing line to drive like a pro.

## Design

**Per-tick computation:**
1. Look up optimal speed at current distance from speed profile
2. Look up optimal lateral position from racing line
3. If current speed > optimal: throttle = 0 (lift/brake — physics handles deceleration)
4. If current speed < optimal: throttle = 1.0 (full power)
5. If current speed ≈ optimal: throttle = 0.7 (maintenance)
6. Adjust for conditions: rain → reduce profile by 10-30%, worn tires → reduce by 5-15%

**Tire/condition adjustments:**
- `condition_factor = 1.0 - (tire_wear * 0.15) - (wetness * 0.25) - (damage * 0.1)`
- Applied to the speed profile target: `adjusted_target = profile_speed * condition_factor`
