# T24.3 — Powertrain Efficiency

Implement efficiency computation for engine_map and gearbox.

**Engine map:** efficiency = min(requested_torque, traction_available) / traction_available. If you request more torque than grip allows, excess causes wheelspin (tire wear) but no speed gain.

**Gearbox:** efficiency = torque_curve(actual_RPM) / torque_curve(optimal_RPM). Wrong gear = RPM outside the peak torque band = less power = lower efficiency. The optimal RPM is speed-dependent.

Both must show 0.5-1.5s individual sensitivity in the test script.
