# T21.1 — New Car Template

Create `car_template_v2.py` — the new template players copy to build their car. Organized by physical location on the car (powertrain, hybrid, chassis, pit wall). Each section has a codeable part function with full docstring explaining the PART, its inputs, valid outputs, and what bad code does.

Hardware spec selection at the top. All 10 part functions with working defaults. Strategy function at the bottom.

The file IS the car. Opening it should feel like opening the hood.
