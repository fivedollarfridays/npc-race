# Objective

Bump version to 1.0.0, build the package, test on TestPyPI, then publish to PyPI. Verify the package installs and runs correctly from PyPI in a clean venv.

# Implementation Plan

1. Bump version to 1.0.0 in pyproject.toml
2. Build: `python -m build`
3. Upload to TestPyPI: `twine upload --repository testpypi dist/*`
4. Test install from TestPyPI in clean venv
5. Upload to PyPI: `twine upload dist/*`
6. Verify: `pip install npc-race && npcrace init && npcrace run` in clean venv

# Acceptance Criteria

- [ ] Version is 1.0.0 in pyproject.toml
- [ ] Package builds successfully with `python -m build`
- [ ] Package uploads to TestPyPI without errors
- [ ] Package installs from TestPyPI and runs
- [ ] Package uploads to PyPI without errors
- [ ] `pip install npc-race && npcrace init && npcrace run` works in clean venv
