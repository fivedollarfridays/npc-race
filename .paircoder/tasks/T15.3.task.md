# T15.3 — Sector Markers + DRS Zone Overlays

## Objective

Add visual markers on the track for sector boundaries (S1/S2/S3 lines) and DRS activation zones (blue tint overlay).

## Design

**Sector markers:** Dashed lines across the track at 33% and 66% distance. Small "S1"/"S2"/"S3" labels.

**DRS zones:** Semi-transparent blue overlay on track segments where DRS is active. Data comes from `replay.drs_zones` (if present) or computed from track data.

## Changes to `viewer/js/track-renderer.js`

Add:
```javascript
function drawSectorMarkers(ctx, track, distances, trackLength, trackWidth) {
  // Draw dashed lines at 33% and 66% of track length
}

function drawDRSZones(ctx, track, distances, trackLength, trackWidth, drsZones) {
  // Draw blue semi-transparent overlay on DRS zone segments
}
```

Call from the track rendering pipeline.
