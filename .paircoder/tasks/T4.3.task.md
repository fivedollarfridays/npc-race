# Objective

Update bot_scanner to allow json import and controlled file access for car data persistence.

# Implementation Plan

1. Add "json" to ALLOWED_IMPORTS
2. Remove "open" from BLOCKED_CALLS
3. Add path validation: open() only allowed for paths matching cars/data/*.json pattern
4. Block path traversal (.. in path)

# Acceptance Criteria

- [ ] import json passes scan
- [ ] open("cars/data/mycar.json", "r") passes scan
- [ ] open("cars/data/mycar.json", "w") passes scan
- [ ] open("/etc/passwd") fails scan
- [ ] open("cars/data/../../etc/passwd") fails scan
- [ ] open("cars/../secrets.json") fails scan
- [ ] All existing car files still pass scan
