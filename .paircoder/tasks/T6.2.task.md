# T6.2 — Speed Recalibration

## Objective

Fix the speed explosion. Cars currently reach 2,000-2,700 km/h and lap Monza in 7 seconds. Target: top speed ~340 km/h, Monza lap ~80-85 seconds, Monaco ~72-78 seconds. The simulation must feel like F1 — fast straights, hard braking, slow corners.

## Root Cause Analysis

The speed explosion comes from:
1. `POWER_SPEED_FACTOR = 90` — a car with power=6 gets base_max_speed = 155 + 540 = **695 km/h**
2. No aerodynamic drag model — speed is only limited by target_speed, not by v² drag
3. Draft bonus adds speed beyond target_speed (no cap)
4. `world_scale = track_length / 3333.0` — calibration constant disconnected from real lengths

## Implementation Plan

### Recalibrate constants in `engine/physics.py` (from T6.1)

**Speed formula target:** For a balanced car (power=20/40=0.5), base_max_speed ≈ 310 km/h. For max power car (40/40=1.0), ≈ 345 km/h.

```python
BASE_SPEED = 260           # was 155. km/h floor for zero-power car
POWER_SPEED_FACTOR = 85    # was 90. But net effect smaller: 260 + 1.0*85 = 345 km/h max
WEIGHT_SPEED_PENALTY = 30  # was 60. Reduce variance: heaviest car loses 30 km/h
```

**Corner speed:** Slow corners 60-80 km/h, fast sweepers 200-250 km/h.
```python
GRIP_BASE_SPEED = 65       # was 60. Minimum corner speed
GRIP_SPEED_RANGE = 200     # was 300. Max corner speed = 65 + 200 = 265 km/h
CURVATURE_FACTOR = 50.0    # was 47.0. Slightly sharper cornering response
```

**Acceleration:** 0-200 km/h in ~4s, 0-300 km/h in ~10s (real F1).
```python
ACCEL_BASE = 30            # was 50
ACCEL_POWER_FACTOR = 35    # was 60
```

**Braking:** Strong braking to make corners feel real.
```python
BRAKE_BASE = 200           # was 80. Much stronger — F1 brakes are immense
BRAKE_FACTOR = 150         # was 100
```

### Add aerodynamic drag

New function in `engine/physics.py`:
```python
DRAG_COEFFICIENT = 0.0003  # v² drag: at 300 km/h, drag = 27 km/h/s

def apply_drag(speed: float, dt: float) -> float:
    """Apply v-squared aerodynamic drag. Returns speed after drag."""
    drag = speed * speed * DRAG_COEFFICIENT * dt
    return max(0.0, speed - drag)
```

Call `apply_drag()` every tick in `_apply_physics()` AFTER acceleration, BEFORE position update.

### Add hard speed cap

```python
MAX_SPEED = 370.0  # km/h — absolute ceiling (drag handles most cases)
```

### Fix world_scale

Replace `track_length / 3333.0` with calibration based on `real_length_m`:
```python
# If real_length_m is provided, use it to compute world_scale
# so that distance in sim units maps to real meters correctly
if real_length_m:
    world_scale = track_length / real_length_m
else:
    world_scale = track_length / 3333.0  # fallback for procedural tracks
```

This ensures that `speed_km/h * (1/3.6) * world_scale * dt` produces correct real-world distances.
