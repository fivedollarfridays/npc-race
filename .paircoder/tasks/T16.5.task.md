# T16.5 — CLI Command + Season Output

## Objective

Add `npcrace season` CLI command that runs a championship season. Options for preset or custom calendar, laps, output directory.

## CLI Design

```
npcrace season                        # run "short" season (5 races)
npcrace season --calendar full        # run "full" season (10 races)
npcrace season --tracks monza,monaco  # custom 2-race season
npcrace season --laps 3               # 3 laps per race
npcrace season --output-dir results/  # save replays here
```

## Changes

### `cli/commands.py`
Add `cmd_season(args)` function that calls `run_season()`.

### `cli/main.py`
Add `season` subparser with `--calendar`, `--tracks`, `--laps`, `--output-dir` arguments.

## Tests

- `test_season_cli_exists` — `npcrace season --help` works
- `test_season_cli_short` — runs short season without error
- `test_season_output` — creates output directory with replay files

## Acceptance Criteria

- `npcrace season` command works
- 3 new tests pass
- All existing CLI tests pass
