# Objective

Create procedural sound effects using Web Audio API: engine tone, tire squeal, downshift pops, aero whoosh, and crowd ambience. No external audio files.

# Implementation Plan

1. Create `viewer/js/sound-engine.js` with `SoundEngine` class
2. Initialize AudioContext on first user interaction (click play or unmute) to comply with browser autoplay policy
3. Engine tone: sawtooth oscillator + 2nd harmonic, base frequency 80-400Hz mapped to leader speed (0-300 km/h). Gain envelope follows speed changes
4. Tire squeal: white noise through bandpass filter (2-4kHz), gain = curvature * speed * factor, only audible above threshold
5. Downshift pops: short noise burst (50ms) triggered when speed drops > 20 km/h between frames
6. Aero whoosh: brown noise (filtered white noise through lowpass), gain proportional to leader speed
7. Crowd ambience: brown noise loop at low volume, gain spikes briefly on overtake events
8. Volume control: range slider (0-100%), stored in SoundEngine state
9. Mute button: toggles master gain to 0, shows speaker icon state
10. All sounds stop when paused, resume smoothly when playing (no pops/clicks)

# Acceptance Criteria

- [x] Engine pitch audibly rises and falls with leader speed
- [x] Tire squeal audible in corners (high curvature + high speed)
- [x] Downshift pops on hard braking
- [x] Aero whoosh volume scales with speed
