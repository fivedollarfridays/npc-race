# T46.1 — Ghost Level System
Create `engine/ghost.py` — defines 5 ghost levels with calibrated flaws.

## Levels
1. Over-revs (shifts at 14000 RPM) — player wins easily
2. Default settings (12800 RPM shift) — player must improve gearbox
3. Optimal gearbox, bad cooling (effort=1.0) — player learns cooling
4. Optimal gearbox+cooling, no pit strategy — player learns pits (3 laps)
5. Tortoise (real rival car) — graduation race

## API
- `create_ghost(level: int) -> dict` returns car dict compatible with PartsRaceSim
- `GHOST_LEVELS` dict with metadata (name, flaw, description, laps)
