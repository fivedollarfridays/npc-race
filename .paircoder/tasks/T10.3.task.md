# T10.3 — Simulation Integration: Weather Wiring

## Objective

Wire the weather model into simulation.py. CRITICAL: simulation.py is at 15/15 functions, 389/400 lines. NO new functions. All logic delegated to weather_model.py calls.

## CONSTRAINT

simulation.py: 389 lines / 15 functions. Hard limits: 400/15.
Strategy: add ~5-8 lines of import+call wiring distributed across existing functions.

## Implementation Plan

### Imports (top of simulation.py)

```python
from .weather_model import (create_weather_state, update_weather,
                             get_wetness_grip_mult, get_wetness_wear_mult,
                             generate_forecast)
```

### RaceSim.__init__ — add weather state

Add parameter and initialization:
```python
def __init__(self, ..., weather_enabled=True):
    ...
    self.weather = create_weather_state()
    self.weather_enabled = weather_enabled
    self._weather_forecast = []
```

### step() — update weather on lap change

After the safety car update block, add:
```python
# Update weather on leader lap change (same hook as SC)
if leader_lap_changed and self.weather_enabled:
    self.weather = update_weather(self.weather, self.rng)
    self._weather_forecast = generate_forecast(self.weather, 5, self.rng)
```

Note: `leader_lap_changed` is already computed for SC updates. Reuse it.

### _apply_physics() — wetness grip penalty

In the grip chain (where tire_grip * temp_grip * dirty_grip are multiplied):
```python
wetness_grip = get_wetness_grip_mult(
    self.weather.get("wetness", 0.0), state.get("tire_compound", "medium")
) if self.weather_enabled else 1.0
```
Then multiply into the grip chain.

### _apply_tire_wear() — wetness wear modifier

After the SC modifier:
```python
if self.weather_enabled:
    throttle_factor *= get_wetness_wear_mult(
        self.weather.get("wetness", 0.0), state.get("tire_compound", "medium"))
```

### build_strategy_state() — expose weather fields

Add to return dict:
```python
"track_wetness": self.weather.get("wetness", 0.0) if self.weather_enabled else 0.0,
"weather_forecast": self._weather_forecast if self.weather_enabled else [],
"weather_state": self.weather.get("state", "dry") if self.weather_enabled else "dry",
```

### Replay enrichment

In replay.py `record_frame()`, add:
```python
"track_wetness": round(state.get("_track_wetness", 0.0), 3),
```

In simulation.py step(), propagate wetness to car states before _record_frame:
```python
for s in self.states:
    s["_track_wetness"] = self.weather.get("wetness", 0.0)
```

### engine/__init__.py

Add weather exports:
```python
from .weather_model import (create_weather_state, update_weather,
                             get_wetness_grip_mult, get_wetness_wear_mult,
                             generate_forecast, get_optimal_compound)
```

### race_runner.py

Pass `weather_enabled` through to RaceSim if provided. Default True.
