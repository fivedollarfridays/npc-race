# T24.5 — Braking + Diff Efficiency

**Brake bias:** efficiency = actual_braking_G / max_possible_G. Compute front and rear lockup thresholds separately based on tire grip per axle + weight transfer. The optimal bias depends on speed, grip, and weight distribution. Wrong bias → one axle locks → total braking reduced.

**Differential:** efficiency = actual_corner_exit_speed / max_exit_speed. Over-locked → understeer (can't rotate), slower through corner. Under-locked → inside wheelspin (less traction), slower acceleration on exit. Optimal depends on corner phase and lateral G.

Both must interact: brake_bias affects front tire temps which affects grip which affects differential effectiveness.
