# T5.5 — Seed Cars + Template Update

## Objective

Update `car_template.py` with documentation for all new Sprint 5 fields. Update 3 of the 5 seed cars to demonstrate the new features. All 5 cars must still pass bot_scanner.

## Implementation Plan

### `car_template.py` additions

Document new state fields:
```python
# Tire temperature
state["tire_temp"]             # float °C — current tire surface temp
                               # Optimal: soft ~90, medium ~80, hard ~70
                               # Below optimal = cold, reduced grip
                               # Above optimal = overheating, accelerated wear

# DRS
state["drs_available"]         # bool — can activate this lap
state["drs_active"]            # bool — currently open
state["in_drs_zone"]           # bool — car is in a DRS detection/activation zone
# To use: return {"drs_request": True} when in_drs_zone and gap_ahead_s < 1.0

# Car setup (read-only, set at module level)
state["current_setup"]         # dict — your car's active setup values
```

Document new module-level setup constant:
```python
# Optional SETUP dict — pre-race configuration (neutral defaults if omitted)
SETUP = {
    "wing_angle":    0.0,   # -1.0 low drag, +1.0 high downforce
    "brake_bias":    0.5,   # 0.3 rear to 0.7 front
    "suspension":    0.0,   # -1.0 soft, +1.0 stiff
    "tire_pressure": 0.0,   # -1.0 low, +1.0 high
}
```

### Seed car updates (3 of 5 cars — keep brickhouse and glasscanon unchanged as baseline)

**GooseLoose** — setup + tire temp awareness:
- Add `SETUP = {"wing_angle": -0.2, "brake_bias": 0.55, ...}` (slightly low-drag for power car)
- In strategy: pit slightly earlier if `tire_temp > 105.0` (overheating indicator)

**SlipStream** — DRS + setup:
- Add `SETUP = {"wing_angle": -0.5, ...}` (low drag — drafting + DRS synergy)
- In strategy: `"drs_request": True` when `in_drs_zone and gap_ahead_s < 1.0`

**Silky** — tire temp + setup:
- Add `SETUP = {"wing_angle": 0.4, "suspension": -0.2, ...}` (high downforce corner car)
- In strategy: adjust engine mode based on `tire_temp` — conserve if `tire_temp > 100.0` to let tires cool

### Constraints

- All 5 cars must still pass `npcrace validate cars/<name>.py`
- All updated cars ≤ 100 lines (arch limit)
- No new imports needed — `tire_temp`, `in_drs_zone` etc. come via state dict
