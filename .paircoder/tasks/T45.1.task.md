# T45.1 — Time Trial Engine
Create `engine/time_trial.py` — runs a single car for 1 lap with no rivals, no safe_call threading, no replay recording. Direct function calls for maximum speed.

## Implementation
- `run_time_trial(car_dir, track_name) -> TrialResult`
- Load single car via `load_car_project()`
- Set `safe_call.TIMEOUT_ENABLED = False` for local mode
- Run PartsRaceSim with 1 car, 1 lap, fast_mode=True
- Collect per-part efficiency from call_logs
- Return: lap_time, sector_times, per_part_efficiency dict
- Restore TIMEOUT_ENABLED after run

Target: < 1 second for 1-lap Monza.
