# Objective

Create engine/fuel_model.py with fuel consumption, engine modes, and weight computation.

# Implementation Plan

1. ENGINE_MODES: push (1.2x consumption, 1.05x power), standard (1.0/1.0), conserve (0.8x/0.92x)
2. compute_starting_fuel(laps, track_length_m, base_consumption_per_m) — fuel in kg for race + 5% margin
3. compute_fuel_consumption(throttle, engine_mode, base_rate, dt) — kg consumed per tick
4. compute_weight_from_fuel(fuel_kg, max_fuel_kg) — normalized 0-1 weight penalty
5. get_engine_mode(name) — return mode config, default standard

# Acceptance Criteria

- [x] Starting fuel proportional to laps × track_length
- [x] Consumption scales with throttle (0 throttle = 0 consumption)
- [x] Push mode burns 20% more, conserve burns 20% less
- [x] Weight from fuel decreases monotonically as fuel burns
- [x] Empty tank returns 0 weight penalty
- [x] Module < 120 lines, < 15 functions
