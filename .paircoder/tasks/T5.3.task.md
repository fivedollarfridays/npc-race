# T5.3 — Car Setup Sliders

## Objective

Create `engine/setup_model.py` — a functional module that lets cars declare a pre-race setup configuration. Setup sliders are tradeoffs: high wing gives more downforce (grip bonus) but reduces top speed; stiff suspension heats tires faster but sharpens response. Cars declare setup at module level (like stats); the engine applies physics modifiers.

## Implementation Plan

### New file: `engine/setup_model.py` (~110 LOC)

```python
DEFAULT_SETUP = {
    "wing_angle":    0.0,   # -1.0 (low drag) to +1.0 (high downforce)
    "brake_bias":    0.5,   # 0.0 (rear) to 1.0 (front-heavy)
    "suspension":    0.0,   # -1.0 (soft) to +1.0 (stiff)
    "tire_pressure": 0.0,   # -1.0 (low) to +1.0 (high)
}

SETUP_BOUNDS = {k: (-1.0, 1.0) for k in DEFAULT_SETUP}
SETUP_BOUNDS["brake_bias"] = (0.3, 0.7)  # tighter range

def validate_setup(raw: dict) -> dict:
    """Merge with defaults, clamp all values to SETUP_BOUNDS. Returns clean dict."""

def wing_effect(wing_angle: float) -> tuple[float, float]:
    """Returns (aero_mult, power_mult). High angle: aero +15%, power -8%."""

def brake_bias_effect(bias: float) -> float:
    """Returns brakes multiplier. Optimal ~0.58; deviations reduce braking efficiency."""

def suspension_effect(stiffness: float) -> float:
    """Returns tire_temp_rate_mult. Stiff (+1.0) = 1.3×; soft (-1.0) = 0.8×."""

def tire_pressure_effect(pressure: float) -> tuple[float, float]:
    """Returns (temp_offset_C, grip_mult). High pressure = hotter but lower rolling resist."""

def apply_setup(car_stats: dict, setup: dict) -> dict:
    """
    Return adjusted_stats dict with keys matching car_stats.
    Does NOT mutate car_stats. Used by simulation to compute effective physics values.
    Keys: effective_aero, effective_power, effective_brakes, temp_rate_mult, temp_offset
    """
```

### Car file convention (module-level, like existing stats)

```python
# In a car file:
SETUP = {
    "wing_angle": 0.5,      # high downforce for Monaco
    "brake_bias": 0.6,
    "suspension": -0.3,
    "tire_pressure": 0.0,
}
```

If `SETUP` not defined → `validate_setup({})` → all defaults. Backward compatible.

### Car loader update

`engine/car_loader.py`: read optional `SETUP` dict from car module, run through `validate_setup()`, attach to car state as `setup`.

### Export via `engine/__init__.py`

```python
from .setup_model import validate_setup, apply_setup, DEFAULT_SETUP
```
