# T24.6 — Chassis Efficiency

**Suspension:** efficiency = actual_downforce / optimal_downforce. The ride height sweet spot is speed-dependent. Too low at high speed → bottoming (20% downforce loss). Too high → less ground effect. Only adaptive code that changes with speed hits the optimum.

**Cooling:** efficiency = speed_with_cooling / speed_with_optimal_cooling. More cooling = more drag = lower top speed. Less cooling = engine overheats = power loss (-2%/°C above 120°C). The optimal is a narrow band that depends on engine load and ambient conditions.

These must interact: suspension ride_height affects downforce which affects cooling airflow needed.
