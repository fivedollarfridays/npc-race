# T6.5 — Replay Enrichment with Timing Data

## Objective

Wire timing module (T6.4) into simulation.py and enrich replay frames with timing data. Add gap-to-leader, elapsed time, sector info, and lap times to every frame. Enrich results with best lap, total time, and pit stop count.

## Implementation Plan

### Wire timing into simulation.py

In `RaceSim.__init__()`:
```python
from .timing import create_timing, get_sector_boundaries
self.timings = create_timing([c["name"] for c in cars])
self.sector_boundaries = get_sector_boundaries(track_data if track_data else {})
```

In the per-tick loop (wherever lap/distance is updated):
```python
from .timing import update_timing
timing_result = update_timing(
    self.timings, cs["name"], distance_pct, cs["lap"],
    self.tick, TICKS_PER_SEC, self.sector_boundaries
)
cs["_timing"] = timing_result  # temporary, used by replay
```

### Enrich replay frames (engine/replay.py)

Add to per-frame per-car dict:
```python
"elapsed_s":      round(tick / TICKS_PER_SEC, 2),
"gap_ahead_s":    round(cs.get("gap_ahead_s", 0.0), 2),
"gap_to_leader_s": round(gap_to_leader, 2),
"current_sector": cs["_timing"]["current_sector"],
"last_lap_time":  cs["_timing"].get("lap_time"),  # None unless lap just completed
```

### Enrich results (engine/replay.py)

Add to per-car results dict:
```python
"total_time_s":   round(finish_tick / TICKS_PER_SEC, 2),
"best_lap_s":     timing.best_lap,
"best_lap_num":   timing.lap_times.index(timing.best_lap) + 1 if timing.best_lap else None,
"lap_times":      timing.lap_times,
"pit_stops":      cs["pit_stops"],
```

### Expose gap_ahead_s in strategy state

`gap_ahead_s` already exists in `build_strategy_state()` but isn't in replay frames. Add it to the frame recording.

### Expose timing to strategy state (read-only)

Add to `build_strategy_state()`:
```python
"elapsed_s":     tick / TICKS_PER_SEC,
"last_lap_time": timing.lap_times[-1] if timing.lap_times else None,
"best_lap_time": timing.best_lap,
```

### Console output enhancement

After race results, print timing summary:
```
🏆 RESULTS
────────────────────────────────────

  P2  BrickHouse     1:25.789  +2.333s
  P3  GooseLoose     1:27.012  +3.556s
```
